## 🚀 Highlight Projects

Here are a few projects I've recently worked on that I'm particularly proud of. Feel free to explore the repositories for code samples!

### [Rhayader](https://github.com/chlebicz/rhayader) - Functional Language Interpreter
* **Tech Stack:** C++, CMake, Catch2
* **Key Features:** Implemented a custom tokenizer, parser, evaluator, and several simple built-in modules from scratch.

### [Obiekty sportowe](https://github.com/chlebicz/obiekty-sportowe) - Interactive Map of Sports Facilities in Poland
* **Tech Stack:** NextJS, NestJS, PostgreSQL, Jest, Tailwind, Leaflet
* **Key Features:** Built a data pipeline using web scraping, cleaning, and combining data from multiple sources utilizing AI embeddings. Applied clustering algorithms to efficiently group nearby facilities on the UI.

### [Sudoku Game](https://github.com/chlebicz/sudoku) - Desktop Application
* **Tech Stack:** Java, Maven, JavaFX, JUnit, PostgreSQL, Docker
* **Key Features:** Developed using strict OOP principles, MVC architecture, and Design Patterns. Includes internationalization and database containerization.
